## Ferramentas utilizadas no desenvolvimento do projeto

- VS Code
- Node.js
- SQL Workbench 8.0 CE

## Como rodar o projeto

No terminal execute os comandos nessa ordem ou execute start-project.bat

- cd backend
- npm install
- cd ..
- cd frontend
- npm install
- cd ..
- cd backend
- npm run dev
- cd ..
- cd frontend
- npm start
